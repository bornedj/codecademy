HTML and CSS cheat sheet

This is a cheat sheet for CSS selector concepts.

Selector components

Item Example Description
Selector p Indicates which element the the rule applies too
Declartion {} Follows the selector and contains the rules to be applied
Declartion Statement color: red; The rule to be applied to to the tag

Specificity

Selector Example Description
Tag p Lowest level of specificity. Will apply to all elements with that tag
Class "article-heading" Class is defined by the website's writer. The same class can be used by many tags/elements
id "last-button" Most specific indentifier. Ids are unique indentifiers
child p.span Same level of specificity as a tag. Will apply to all elements that are the child of the tag
Descendant selector body h1 Will format all tags within a specified tag, doesn't neccessarily need to be the child
State selector a:hover Changes the style based on user input/interaction
